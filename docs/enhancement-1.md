# Enhancements

## Startup

1. Application should start with the last used vault selected and opened.

## Vault Explorer

1. Move the `Open Vault` button to the `File` menu.
2. The files should just be labelled with the filename without the extension.
3. Only `.md` files should be included in the explorer.
4. Remove the Rename and Delete buttons from each file entry and replace with a context popup menu.
5. Remove the buttons from each Folder entry and replace with a context popup menu.
6. The vault folder name should have a context popup menu for New Note and New Folder

## Note Editor

1. The edit control should be sized to the full height and width of the panel.
